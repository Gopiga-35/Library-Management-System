# Library Management System

The Library Management System is a Java-based web application designed to
manage library operations such as adding books, viewing book details,
and handling user interactions efficiently.

## Features
- Add new books to the library
- View available book details
- Search books by ISBN

## Technologies Used
- Java
- Servlets & JSP
- JDBC
- HTML / CSS
- Apache Tomcat
- Oracle Database
